## Context
In the previous ADR, we designed the architecture of the data mesh (how it maps to the organization) and the data contracts and governance standards (everything needed to define a high-quality data product). In this ADR, we define how to actually transform raw datasets into high-quality data products, which involves designing in detail the medaillion architecture, dimensional modelling and pipeline tooling that will be used to create high-quality data products.
